## 🚀 Proyectos Destacados (Featured Projects)

### 📦 [Micro-CRM & Sistema de Gestión Comercial](https://github.com/adrianescalona228/flask-business-crm)
Un sistema de administración comercial a medida, control de inventario y CRM diseñado para una distribuidora de repuestos automotrices que opera en producción real.
*   **Automatización de reportes:** Generación automática de notas de entrega Excel (`openpyxl`) integradas con la nube.
*   **Lectura inteligente con IA:** Carga de facturas físicas de proveedores usando modelos OCR de OpenRouter (Gemini 2.5 Flash).
*   **Contenedorización robusta:** Despliegue con Docker Compose y políticas de tolerancia a fallos/cortes eléctricos.
*   *Stack: Python, Flask, SQLite (modo WAL), Docker, JavaScript vanilla.*

### 🤖 Agente de IA para WhatsApp & Automatización de Ventas
Un prototipo avanzado de asistente inteligente para la atención al cliente, filtrado de leads y consulta de stock en tiempo real.
*   **Filtrado inteligente:** Clasificación automática de mensajes comerciales vs. personales usando Modelos de Lenguaje.
*   **Orquestación de flujos:** Diseñado en n8n y conectado mediante Evolution API para respuestas dinámicas a clientes por WhatsApp.
*   **Integración con bases de datos:** Acceso interactivo en tiempo real al inventario de repuestos.
*   *Stack: n8n, Evolution API, LLMs, Webhooks, REST APIs.*
